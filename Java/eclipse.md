# [Eclipse 快捷键](http://www.cnblogs.com/iamfy/archive/2012/07/11/2586869.html)

- 查找

```
ctrl+alt+R

ctrl+alt+T
```

- 编辑

```
alt+上下 
ctrl+alt+上下
Control-Q
格式化 Ctrl+Shift+F 
```

- 重构

```
alt+shift+r：重命名
alt+shift+l以及alt+shift+m：提取本地变量及方法
```

- 阅读代码

```
F3, alt+左右；
文本编辑器 查找上一个 Ctrl+Shift+K ；
文本编辑器 查找下一个 Ctrl+K 
```

- 提示

```
ctrl+1
```
