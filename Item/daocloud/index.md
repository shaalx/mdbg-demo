#	体验创新

##	与其说是创新，不如说是解决了用户的痛点。

*	App重启，数据丢失 ---> Volume来了，对应的docker的 `-v`命令。

	![volume][6]

*	App在运行中的干预，为此还专门写了一个在web端到后台执行命令的工具，不是很好用 ---> 控制台来了。

	![console][7]

*	本来打算开启一个rpc的微服务，结果还不支持rpc，还在开发中，期待吧。
